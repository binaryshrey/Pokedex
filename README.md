## Development setup

```
git clone https://github.com/binaryshrey/Pokedex.git
cd Pokedex
npm i
gatsby develop
```

Development server runs at `http://localhost:8000`.

## Prod setup

```
gatsby build
gatsby serve
```
Production server runs at `http://localhost:9000`.
Build is generated at `public` (which is deployable).
